# 🏙️ App de Coordinación Comunitaria
App para la Gestión Urbana Participativa (Grupo D).

## Módulos y Equipos
* **Equipo A:** Autenticación, Roles y Seguridad.
* **Equipo B:** Interfaz de Usuario y Captura de Incidentes (Flutter).
* **Equipo C:** Inteligencia (NLP) y Notificaciones (Node.js / Firebase).

## Stack Tecnológico
* Frontend: Flutter
* Backend & Serverless: Node.js, Firebase Cloud Functions, Genkit (Gemini)
* Base de datos: Cloud Firestore
